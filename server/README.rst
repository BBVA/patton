What's patton-server?
=====================

patton-server provides a service that resolves CVEs and CPEs from NIST and perform intelligent queries to a postgres database. You can consume the service via api, a commandline client or using the Postman collection provided in this repository.

You can see a demo video running patton-server + patton-cli:
